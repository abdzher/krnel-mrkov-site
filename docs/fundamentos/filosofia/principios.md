# Principios y Valores

El proyecto no solo se rige por código, sino por una ética de trabajo orientada a la excelencia técnica y académica.

### Principios Técnicos
* **Reproducibilidad Absoluta:** "El código es la ley". Cualquier experimento ejecutado en la plataforma debe ser replicable gracias a la inmutabilidad de la configuración de **KRNEL**.
* **Automatización (IaC):** Reducimos el error humano mediante la Infraestructura como Código, permitiendo que el sistema sea resiliente y escalable.
* **Eficiencia de Recursos:** Maximizamos la vida útil y la potencia del hardware institucional mediante una orquestación inteligente de cargas de trabajo.

###  Valores Fundamentales
* **Soberanía Tecnológica:** Fomentamos el uso de software libre y de código abierto para asegurar que la universidad mantenga el control total sobre su infraestructura.
* **Democratización del Cómputo:** El acceso a GPUs y procesamiento de alto nivel no es un privilegio de pocos, sino un derecho de cada investigador y alumno de la comunidad **MRKOV**.
* **Colaboración Transparente:** Promovemos una cultura donde el conocimiento técnico se comparte abiertamente, permitiendo que cada contribución al núcleo de **KRNEL** beneficie a todos.
