# Bienvenido a la documentación del proyecto KRNEL-MRKOV

Las siguientes páginas están dedicadas a documentar de manera técnica la información necesaria para poder contribuir al proyecto. El orden de presentación no es al azar, tiene una estructura secuencial con el objetivo de introducir a los lectores a la parte más ténica del proyecto.

Se espera que el lector primero comprenda los conceptos básicos, estudie las fuentes externas y proceda a entender las decisiones de arquitectura que fueron tomadas, para luego aprender a instalar el sistema y darle mantenimiento por sí mismos sin necesidad de acudir a la automatización, de modo que entienda la lógica detrás.

Posteriormente, pueden acudir a aprender la lógica de la automatización, aprender a administrar el sistema, crear un entorno de pruebas y modificar los valores de configuración. Finalmente se dedica una breve sección a como modificar la presente documentación.
